Highlights

Pure Emacs Lisp

No other language, no other framework. Just Org-Mode, HTML, CSS and Weblorg. Do everything inside Emacs!

Static

Easily hosted on GitHub pages, Amazon S3 or any other HTTP service; builds can be automated via GitHub Actions, Travis or any other CI.

Powerful Templating

with templatel, weblorg can be themed with a language that provides variable substitution and control flow, inspired by Python's Jinja.

Blog Ready

Permalinks, categories, pages, posts, theming and custom templates.
